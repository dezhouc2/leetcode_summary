
```python
time:o(m+n) m为图的边数 n为图的点数
space: o(n)

思路:
【最小的点集使得从这些点出发能到达图中所有点】就是入度为零的所有点的集合 -> 入度为0的点为 不存在一个点y使得 y能到达x(入度为0的点)。

同时因为是有向图所以 图为连通图 -> 每一个点都必有一条路径到达另外一个点 

class Solution:
    def findSmallestSetOfVertices(self, n: int, edges: List[List[int]]) -> List[int]:
        
        result = []
        set_all = set()
        for ele in edges:
            if ele[1] not in set_all:
                set_all.add(ele[1])
            
        
        #non-reachable
        for ele in edges:
            if ele[0] not in set_all:
                result.append(ele[0])
        
        return list(set(result))
```
