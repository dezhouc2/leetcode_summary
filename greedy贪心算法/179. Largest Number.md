思路:
对nums的任意两个数值的顺序 -> 取决于这个两个数值的组合的排序的大小, 如果ab的拼接结果 > ba的话那么就应该让a在b之前 > 可以考虑有什么package可以通过对比数值的结果来排序数值的位置

知识点:
functools.cmp_to_key(function)

当对比a+b : b+a的时候是在str的形式所以先考虑把nums里面的数值转化为str type
方式一:
num = list(map(int, nums)) ##map(转为成, 初始iterable) 就转化成为str type list


class Solution:
    def largestNumber(self, nums: List[int]) -> str:
        
        #since nums[i] contains case of 0
        #base case: nums[0,0]
        
        def cmp(a,b):
            if a+b > b+a:
                return 1
            
            elif a+b < b+a:
                return -1
            
            else:
                return 0
        
        final = list(map(str,nums))
        
        
        final = sorted(final, key = functools.cmp_to_key(cmp),reverse = True)
        if final[0] == "0":
            return "0"
        else:
            return "".join(final)
