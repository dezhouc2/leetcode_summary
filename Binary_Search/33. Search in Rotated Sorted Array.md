
```python

思路:
因为有两段sorted的区间 -> 对两段sorted区间进行考虑

left sorted的判定为 nums[l] <= nums[mid]:
    考虑target 和left sorted区间的关系:
        是否 > nums[mid]
        是否小于这个区间的最小值


right sorted:
    同理


class Solution:
    def search(self, nums: List[int], target: int) -> int:
        
        l,r = 0, len(nums)-1
        
        while l <= r:
            mid = (l+r)//2
            
            if nums[mid] == target:
                return mid
            
            
            #if left hand side is sorted
            elif nums[l] <= nums[mid]:
                if target > nums[mid] or target < nums[l]:
                    l = mid +1
                
                else:
                    r = mid-1
            
            elif nums[l] > nums[mid]:
                if target < nums[mid] or target > nums[r]:
                    r = mid -1
                
                else:
                    l = mid +1
        
        return -1
```
