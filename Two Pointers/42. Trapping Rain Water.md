思路:
通过examples 观察看出 水的面积 = min(max(left),max(right)) - height[i]

如果要用减小空间的算法考虑用双指针

注意:
  怎么知道是l还是r 移动指针? a: leftmax和rightmax对比哪个小移动哪边的指针 目的是为了最大化  水的面积 = min(max(left),max(right)) - height[i]
  
  遍历每一个点要求出当前点的 max(left),max(right)? a: 每次移动的时候都通过max来更新
  

class Solution:
    def trap(self, height: List[int]) -> int:
        
        l,r = 0, len(height)-1
        leftmax,rightmax = height[l],height[r]
        result = 0
        
        while l < r:
            if leftmax > rightmax:
                r = r-1
                rightmax = max(rightmax,height[r])
                result = rightmax - height[r] + result
                
            
            else:
                l = l+1
                leftmax = max(leftmax,height[l])
                result = leftmax - height[l] + result
                

        return result
