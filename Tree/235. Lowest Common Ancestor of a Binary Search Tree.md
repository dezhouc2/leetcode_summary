
```python

知识点:
time:o(n)
space:o(h) -> tree height

--# 二叉搜索树性质:
如果左子树不为空, 则根节点的数值均大于左子树节点的数值

如果右子树不为空, 则根节点的数值均小于右子树节点的数值


--# 思路: (递归)
题目给出了 p,q的数值 和root的数值 结合二叉搜索树的知识点 -> 考虑root.val和 p,q 的数值的大小关系

分类讨论:
1.如果都小于 root.val，则p和q都在左子树, 输出左子树的root即可
2.如果都大于root.val, 输出右子树即可
3.如果在 p < root < q则, 输出root本身

--# 之前卡在递归代码不知道怎么想:
按照分类讨论的思路 输出root.left/root.right 作为结果 -> 这一部分为递归的过程



class Solution:
    def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':
 
            if root.val > p.val and root.val > q.val:
                return self.lowestCommonAncestor(root.left,p,q)
            
            elif root.val < p.val and root.val < q.val:
                return self.lowestCommonAncestor(root.right,p,q)
            
            else:
                return root


```
