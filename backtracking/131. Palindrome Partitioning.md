错误的思路:
之前每一次遍历都是 单个元素的遍历

正确思路:
每一程遍历应该是 单个slice遍历 -> s[0], s[0:2], ...

如果一个list的载体为 string -> 考虑string中分割元素的手法


time: o(n*2^n) n为length of s, each element of s has 2^n possibilities
space: o(n)

class Solution:
    def partition(self, s: str) -> List[List[str]]:
        
        result = []
        curr = []
        
        def backtrack(cur,index):
            if len("".join(cur[:])) == len(s):
                result.append(cur[:])
                return
            
            for i in range(index, len(s)):
                #判断是否为回文
                p = s[index:i+1]
                if p != p[::-1]:
                    continue
                
                cur.append(s[index:i+1])
                backtrack(cur,i+1)
                cur.pop()
        
        
        backtrack(curr,0)
        return result
