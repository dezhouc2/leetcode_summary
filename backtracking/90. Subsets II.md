q:为什么这道题要排序而 类似的题目78不用排序
a: 如果[2,2,1] 第一个2递归要取2的话 肯定会有一个子集取到1 这不是答案所要的; 对比78,


注意:
这道题得先让数组排序 -> 否则会取到重复的子集

错误的点:
这道题之前想的是用hashset; 可以通过for loop中来判断是否存在相同的数从而来跳过


class Solution:
    def subsetsWithDup(self, nums: List[int]) -> List[List[int]]:
        
        result = []
        curr = []
        nums.sort()
        
        def backtrack(cur,index):
            result.append(cur[:])
            
            for i in range(index,len(nums)):
            
                # 同一层中每一个数和前面的数判断是否一致 -> 如果有就跳过
                if index != i and nums[i] == nums[i-1]:
                    continue
                
                cur.append(nums[i])
                backtrack(cur,i+1)
                cur.pop()
                
            
        
        backtrack(curr,0)
        return result
