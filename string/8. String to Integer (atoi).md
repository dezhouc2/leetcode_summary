这道题用到的知识点:
string.lstrip([characters])

1.if no argument provided,all leading whitespeaces are removed(如果没有输入任何东西的话 就把所有开头的空格removed)
2.所有的characters从左边开始removed,一直到没有match

example:
random_string = '   this is good '
print(random_string.lstrip('s ti')) #his is good 从空格开始removed s没有s 就removed " "然后一直到h是第一个没有在character里面的


string.strip([characters]) #和上面的用法一样 唯一不一样是从左边和右边一起开始removed

example:
string = '  xoxo love xoxo   '
print(string.strip(' xoe')) #lov -> 因为从左边和右边都开始减少" xoe"直到左边第一个 "l"不在" xoe"的范围 和右边的"v"不在 " xoe"的范围
