class Solution:
    def longestPalindrome(self, s: str) -> int:
        
        hashset = set()
        result = 0
        
        for ele in s:
            if ele in hashset:
                result = result +1
                hashset.remove(ele)
            else:
                hashset.add(ele)
        
        if hashset:
            return result*2+1
        else:
            return result*2

time:o(n)
space:o(n)

思路:
根据examples 回文串有两种 一种是偶数 另外一个种是奇数

偶数的话考虑有多少对 -> 用什么方式去计算对? -> 用hashset -> 如果是偶数的话hashset最后没有任何元素输出 numbers of pair*2 

奇数的话是有偶数加一个中间的回文分割 即 b|b 输出numbers of pair*2+1
