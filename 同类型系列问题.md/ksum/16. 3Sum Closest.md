
```python
time:o(n^2)
space:o(n)

思路:
在遍历只剩下两个数值的过程中:
  如果3个数值的合为target 说明是最好的情况
  
无限逼近情况:
  需要变量: 最后输出的值, 定义"无限逼近"的数值(abs(summ-target)),当前的保持最小的"无限逼近"的数值
  
  怎么确认可以获取无限逼近的数值?
    定义"无限逼近"的数值
    if "无限逼近"的数值 小于 当前的保持最小的"无限逼近"的数值:
      更新 当前的保持最小的"无限逼近"的数值
      更新最后的输出数值


class Solution:
    def threeSumClosest(self, nums: List[int], target: int) -> int:
        
        result = float("inf")
        nums.sort()
        final = "inf"
        
        for i in range(len(nums)-2):
            if i > 0 and nums[i] == nums[i-1]:
                continue
            
            l = i+1
            r = len(nums)-1
            
            while l < r:
                summ = nums[i] + nums[l] + nums[r]
                val = abs(summ-target)
                
                if val < result:
                    result = abs(summ-target)
                    final = summ
                
                if summ == target:
                    return summ
                
                elif summ < target:
                    l = l+1
                
                else:
                    r = r - 1
        
        return final
```
