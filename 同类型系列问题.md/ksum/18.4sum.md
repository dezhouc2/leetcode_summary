time: o(n^3) 如果是ksum的话 -> o(n^(k-1))
space:o(n)

思路:
把4sum ->3sum -> 2sum2

先找出前两个数值:
  去重
  一样的操作给前两个数值 -> 考虑dfs


再找出后面两个数值:
  利用two sum2

考虑细节:
1.target的数值剩下多少

2.从哪个index开始找第一个数值(如果是4sum)
k = 4
len(nums) = 6, [0,1,2,3,4,5]
第一个数值的index范围为 [0,1,2] 因为要留出空间给其他3个数

3.当前k剩下的数值 即k剩下为多少

class Solution:
    def fourSum(self, nums: List[int], target: int) -> List[List[int]]:
        
        result, qua = [],[]
        nums.sort()
        
        def foursum(k,start,target):
            if k !=2:
                for i in range(start, len(nums)-k+1):
                    if i > start and nums[i] == nums[i-1]:
                        continue
                    
                    qua.append(nums[i])
                    foursum(k-1,i+1,target-nums[i])
                    qua.pop()     
                return
            
            
            #decrease into 2 nums
            l = start
            r = len(nums)-1

            while l < r:
                if nums[l] + nums[r] == target:
                    result.append(qua+[nums[l],nums[r]])
                    l = l +1
                    while l < r and nums[l] == nums[l-1]:
                        l = l+1


                elif nums[l] + nums[r] < target:
                    l = l +1

                else:
                    r = r- 1
                        
        
        foursum(4,0,target)
        return result
