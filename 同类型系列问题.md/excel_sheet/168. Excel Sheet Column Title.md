做错的原因:
在while loop里面 没有把 columnNumber = columnNumber - 1

为什么？因为 0...25对应的是 "A" -"Z", 每次append的字母都得预先处理-1 因为题目假设了A为1 而不是A为0

--*:背景:
prequiste ord("A") + 1 = ord("B")

0 ... 25  对应
A ... Z


class Solution:
    def convertToTitle(self, columnNumber: int) -> str:
        result = []
        
        #28/26 = 1, 28 mod 26 = 2
        #how to convert digits to alpha -> ord("")
        
        while columnNumber > 0:
            columnNumber = columnNumber - 1
            alpha = (columnNumber % 26)+ ord("A")
            result.append(chr(alpha))
            columnNumber = columnNumber//26
        
        result = "".join(result[::-1])
        return result
