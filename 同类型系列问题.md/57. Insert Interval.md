

```python
time:o(n)
space:o(n)

思路:
intervals 分为重叠和不重叠的区间:

重叠的区间: 可以通过不重叠的区间negation 来判定 即 newinterval[1] >= intervals[i][0] 同时得更新newinterval
注意:
这里更新newinterval之后不可以直接加入result 因为后面可能还会更新, 要一直遍历到不会更新重叠的区间之后再加入到result (通过example2来辨别)


不重叠的区间: 通过newinterval[1] < intervals[i][0] 来判定不重叠然后加入result


class Solution:
    def insert(self, intervals: List[List[int]], newInterval: List[int]) -> List[List[int]]:
        
        #interval problem -> draw picture/diagram
        
        #base case
        if not intervals:
            return [newInterval]
        
        result = []
        i = 0
        
        while i < len(intervals) and intervals[i][1] < newInterval[0]:
                result.append(intervals[i])
                i = i+1
        
        
        while i < len(intervals) and newInterval[1] >= intervals[i][0]:
                newInterval[0] = min(newInterval[0], intervals[i][0])
                newInterval[1] = max(newInterval[1], intervals[i][1])
                
                i = i+1
        result.append(newInterval)
        
        while i < len(intervals) and intervals[i][0] > newInterval[1]:
                result.append(intervals[i])
                i = i+1
        
        return result
```
