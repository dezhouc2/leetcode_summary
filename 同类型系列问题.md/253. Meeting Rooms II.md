time: o(n*logn)
space:o(n)

思路:
numbers of rooms取决于在某一个载体中intervals的数量 -> 考虑用什么载体来存储rooms？(想到类似题目用的是list)

每一个interval和另外一个interval进行比较 -> 如果目前的intervals 大于 另外一个intervals最后一个数值那么把另外一个interval给删除 -> 考虑什么载体可以自由删除元素(stack,queue,heap)

每一个interval的前段部分对比的是 目前载体中的最小值以便判断是否需要额外的房间 所以这个载体用的是 min heap




class Solution:
    def minMeetingRooms(self, intervals: List[List[int]]) -> int:
        
        # 0..................30
        #    5...10
        #           15...20
        
        
        # 2..4
        #      7..10
        
        
        intervals.sort(key = lambda x:x[0])
        
        hp =[]
        hp.append(intervals[0][1])
        
        for i in range(1, len(intervals)):
            
            if intervals[i][0] < hp[0]:
                heapq.heappush(hp, intervals[i][1])
            
            else:
                heapq.heappop(hp)
                heapq.heappush(hp, intervals[i][1])
        
        return len(hp)
