思路:
time: O(log 10(n)) 因为每一次循环都除以10, 即数次结束整个循环 = 每一次除以10*数值大小

##构造一样数值的模板
result = 0

while 当前数值>0:
  最后一个数值 = 当前数值%10
  result = result*10 + 最后一个数值
  当前数值 = 当前数值//10



class Solution:
    def isPalindrome(self, x: int) -> bool:
        
        #via example2
        if x < 0:
            return False
        
        y = x
        result = 0
        
        while x > 0:
            curr = x%10
            result = result*10 + curr
            x = x//10
        
        if y == result:
            return True
        
        else:
            return False
