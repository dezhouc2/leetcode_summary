time:o(n) numbers of cells in isConnected
time:o(n) # elements in queue + # elements in set()

思路:
抽象成 图的邻接矩阵 -> 从没有在hashset中的元素开始遍历(即从0开始)

这道题的难点是怎么把 二维的list 抽象成图 然后再利用图的bfs来解决

class Solution:
    def findCircleNum(self, isConnected: List[List[int]]) -> int:
        
        n = len(isConnected)
        count = 0
        viz = set()
        
        
        def bfs(row):
            queue = []
            queue.append(row)
            viz.add(row)
            
            while queue:
                r = queue.pop(0)
                
                for j in range(n):
                    if isConnected[r][j] == 1 and j not in viz:
                        queue.append(j)
                        viz.add(j)
                    
                
        
        for i in range(n):
            if i not in viz:
                bfs(i)
                count = count+1
        
        return count
