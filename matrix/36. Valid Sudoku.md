
```
time:o(n^2)
space:o(k) space sum of three sets

思路:
对于每一个情况设置一个set, 然后每一次分别加入在matrix里面的元素

知识点:
defaultdict(set) 和 set()的区别:
defaultdict(set) 为一个字典, key的value为一个set()

set(): 里面只是value而已


注意:
在每一个boxes的情况, boxes为 i//3*3 + j//3 这里比较难想

class Solution:
    def isValidSudoku(self, board: List[List[str]]) -> bool:
        
        #0 represents row
        #1 represnts the column
        #2 represnts the boxes
        dic = {0:defaultdict(set), 1:defaultdict(set), 2:defaultdict(set)}
        n = len(board)
        
        for i in range(n):
            for j in range(n):
                if board[i][j] == ".":
                    continue
                
                if board[i][j] in dic[0][i] or board[i][j] in dic[1][j]:
                    return False
                
                square = i//3*3 + j//3
                
                if board[i][j] in dic[2][square]:
                    return False
                
                dic[0][i].add(board[i][j])
                dic[1][j].add(board[i][j])
                dic[2][square].add(board[i][j])
        
        
        return True
```
