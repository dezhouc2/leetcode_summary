思路:
in-place 暗示了用双指针

通过example1 可以想到用两个指针 一个fast, 一个slow
考虑fast的移动规律 -> 观察到当前的值为0的话就递增fast
考虑slow的移动规律 -> 如果当前fast指针对应的数值不为0, fast,slow的指针对应的数值交换且 递增fast,slow指针

time:o(n)
space:o(1)

class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        
        #in-place -> two pointers
        fast = 0
        slow = 0
        
        while fast < len(nums):
            if nums[fast] == 0:
                fast = fast +1
            
            else:
                nums[fast],nums[slow] = nums[slow],nums[fast]
                fast = fast+1
                slow = slow+1
