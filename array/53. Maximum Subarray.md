
```python
time:o(n)
space:o(1)

思路: 贪心算法
一个变量来记得subarray的数值
另外一个变量来记录所有subarray的最大数值

多做几次


class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        
        maxsum = float("-inf")   
        result = 0
        
        for ele in nums:
            result = max(ele, result+ele)
            maxsum = max(result,maxsum)
        
        return maxsum
```
