## 知识点: 异或操作
### 解决: 找唯一问题, 重复问题等
- If we take XOR of zero and some bit, it will return that bit
  - a ^ 0 = a
- If we take XOR of two same bits, it will return 0
  - a ^ a = 0
  - a ^ b ^ a = a ^ a ^ b = 0 ^ b = b

time:o(n)
space:o(1)

class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        
        a = 0
        
        for ele in nums:
            a = a^ele
        
        
        return a
