思路:
time:o(n)
space:o(1)

每一次的profit只能从 买进去之后的日子找

maximum profit = 最大的价格卖出(保证这个价格在最小的买进去的日期之后) - 最小的买进去价格

最大的价格卖出得先遍历这个数组再用max来记录

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        
        profit = 0
        low = float("inf")
        
        for ele in prices:
            low = min(ele,low)
            profit = max(ele-low,profit)
        
        return profit
            


景象问题: 122. Best Time to Buy and Sell Stock II
思路:
通过example1 知道: 计数是通过prices[i] - prices[i-1]来获取的, 如果这个profit的值大于0的话就可以收下即可
example2: 也可以通过计数一个一个来获取profit

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        
        profit = 0
        
        for i in range(1, len(prices)):
            profit = profit + max(prices[i]-prices[i-1],0)
        
        return profit
