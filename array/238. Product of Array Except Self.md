注意:

得出左边的所有数字的乘积 但是落实在代码上有点卡

把右边的base case考虑了即先用1来代替 然后再落实在代码上


思路:
每一个数字的final result都是由于这个数字的左边所有数字和 右边所有数字的乘积

考虑先建立每一个数字的左边的数值 然后再在这个基础上乘上每一个数字所有的右边数值

time:o(n)
space:o(1)

class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        
        result = [0]*len(nums)
        
        #base case
        result[0] = 1
        
        for i in range(1, len(nums)):
            result[i] = result[i-1]* nums[i-1]
        
        
        right =1
        
        for j in reversed(range(len(nums))):
            result[j] = result[j] *right
            
            right = right*nums[j]
        
        
        return result
