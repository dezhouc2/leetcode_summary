思路:
通过example1: 
考虑怎么得到 “A”的数值 -> 思考有什么数值是和字母有关系的? (ord("alphabet")) -> 1 = ord(“A”) - ord("A") + 1即可

看到example2 -> 考虑怎么获得 28的数值 -> 1*sth + 2 = 28 所以 sth 只能等于 26

time:o(n)
space:o(1)

class Solution:
    def titleToNumber(self, columnTitle: str) -> int:
        
        # by example, get principle of element = count * 26 + valelement
        
        count = 0
        
        for ele in columnTitle:
            val = ord(ele) - ord("A")+1
            count = count*26 + val
        
        return count
