time: o((n-k)logn) 每一个pop要 (logn) 一共(n-k)次 
space:(n)

注意:
__init__ 中的k 和 nums不可以直接用都必须先定义

思路:
先考虑用maxheap 还是minheap

[5,6,7] 3rd largest = 5 即 1st smallest

根据题目意思找到kth largest 等价于 第一个最小的值 -> 用minheap 但首先得保证heap中的长度如果大于k的话得pop 直到有k个size


class KthLargest:

    def __init__(self, k: int, nums: List[int]):
        self.size = k
        self.hp = []
        for ele in nums:
            heapq.heappush(self.hp,ele)
        
        while len(self.hp) > self.size:
            heapq.heappop(self.hp)
            

    def add(self, val: int) -> int:
        heapq.heappush(self.hp,val)
        while len(self.hp) > self.size:
            heapq.heappop(self.hp)
        
        return self.hp[0]
