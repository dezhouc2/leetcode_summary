

```python

time:o(n)
space:o(n)

思路:
简单情况当 n= 0的时候, 即可tasks的长度

求最短的个数 -> 考虑是否可以通过找规律来求出 -> 观察到example

最短个数 = 2 个patterns + 一个a + 一个 b 这个等价于 patterns的个数*patterns长度 + 频率最多的字母

patterns个数 = tasks中频率最多的字母的个数 -1; patterns长度 = n+1



class Solution:
    def leastInterval(self, tasks: List[str], n: int) -> int:
        
        #consider numbers of pattern * length of pattern + numbers of max values in counter
        count = Counter(tasks)
        max_ = max(count.values())
        
        output = (max_-1)*(n+1)
        
        for key in count:
            if count[key] == max_:
                output = output +1
        
        return max(output,len(tasks))
```
