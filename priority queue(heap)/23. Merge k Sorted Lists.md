
```python
time: o(n*logn)
space:o(n)

思路:
从所有的list中所有的元素 选取最小的元素作为头接点, 然后(每次选取最小的元素)依此类推形成一个从小到大的list
这里提示了用到heap的结构(每次选取最小的元素); 考虑用一个伪节点作为这个最小元素的上一个节点

listnode的注意点:
while 遍历; 移动节点指针
把元素转化为节点元素


# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def mergeKLists(self, lists: List[Optional[ListNode]]) -> Optional[ListNode]:
        
        heap = []
        
        for i in lists:
            while i:  ##这里注意遍历节点的时候用while 不可以用for
                heapq.heappush(heap,i.val)
                i = i.next  ##移动节点指针
        
        node = vertex = ListNode()
        
        while heap:
            ele = heapq.heappop(heap)
            vertex.next = ListNode(ele)  ##注意把元素转化为节点元素
            vertex = vertex.next
            
        
        return node.next
```
