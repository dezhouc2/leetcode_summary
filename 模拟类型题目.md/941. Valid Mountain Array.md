
```python
time:o(n)
space:o(1)

思路:
模拟向上 和向下

向上:
这个向上和向下的运动过程用 while loop 限制是 i+1 < len(arr) 
q: 为什么不是 i < len(arr)
a: 因为如果是i 的话要对比 i和i+1 就没办法对比, 如果想要对比i 和i+1 那么i+1就是最后的限制


class Solution:
    def validMountainArray(self, arr: List[int]) -> bool:

        i = 0
        
        while i+1 < len(arr) and arr[i] < arr[i+1]:
            i = i+1
            
        if i == 0 or i == len(arr) - 1:
            return False
        
        
        while i+1 < len(arr) and arr[i] > arr[i+1]:
            i = i +1
        
        if i == len(arr)-1:
            return True
        else:
            return False
```
