
## 150. Evaluate Reverse Polish Notation

```
class Solution:
    def evalRPN(self, tokens: List[str]) -> int:
        
        stack =[]
        
        for ele in tokens:
            if ele not in "+/-*":
                stack.append(int(ele))
            
            else:
                result = 0
                a = stack.pop()
                b = stack.pop()
                
                if ele == "+":
                    result = a+b
                elif ele == "-":
                    result = b-a
                elif ele == "*":
                    result = a*b
                
                else:
                    result = int(b/a)
            
                stack.append(result)
        
        return stack.pop()
            

Input: tokens = ["4","13","5","/","+"]
Output: 6
Explanation: (4 + (13 / 5)) = 6

通过这个例子我们知道第二个pop的元素 除以第一个pop的元素

time:o(n)
space:o(n)
      
```
