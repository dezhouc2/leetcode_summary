```python
time:o(n)
space:o(n)

思路:
每一个完整的结构有两个部分:
    先考虑用左边部分进入stack

    当遍历右边的部分看是否为最后一个符号的对应的部分 或 对应的部分没有就false
        注意这里另外一个情况为如果继续的话说明一个完整的结构结束了 得pop stack中的一个部分
    
    


class Solution:
    def isValid(self, s: str) -> bool:
        dc = {"(":")","{":"}","[":"]"}
        stack = []
        
        for ele in s:
            if ele in dc:
                stack.append(ele)
            
            else:
                if not stack or dc[stack[-1]] != ele:
                    return False
                
                stack.pop()
        
        if stack:
            return False
        
        return True
   
time:o(n)
space:o(n)
```
